Directive

Directive used to build pagination control

Markup
 <spsui-pagination-footer
    size-options="[25,50,100]"
    api="ctrl.paginationApi"
    on-ready="ctrl.paginationReady"
    on-change="ctrl.paginationChange">
 </spsui-pagination-footer>
Arguments
Name Type Details
size-options array Array of number for items per page
api object This object is decorated with pagination methods used to change the state. Silent flags can be set to avoid the triggering of an on-change callback
  • nextPage(bool silent)
  • prevPage(bool silent)
  • setPageSize(int newPageSize, bool silent)
  • setPageNumber(int newPageNumber, bool silent)
  • setTotalItems(int newTotal, bool silent)
api.state object This object is read-only and will throw errors if you try to set anything on it
  • pageNumber
  • startIndex
  • endIndex
  • totalItems
  • totalPages
on-ready function Callback function that fires after the component has decorated the api. This lets you know that you can use the methods
on-changed function Callback function that fires when the page changes and receives new and previous state objects
